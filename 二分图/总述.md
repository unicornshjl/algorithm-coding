## 一、二分图的定义与等价性质

- **定义** : 若图的所有顶点可分成两个不相交的集合 $A$ 和 $B$，并且每条边都仅连接 $A$ 与 $B$ 之间的点（即集合内部无边），则该图为**二分图**。
- **等价条件**：
  1. 图中不存在**奇数环**；
  2. 图可以被正确地**两种颜色染色**（相邻点颜色不同）；
  3. 图是一个**无奇环的连通图**。

因此：
$$
图为二分图  ⟺  不存在奇数环  ⟺  可以用染色法染色
$$

## 二、判断是否为二分图（染色法）

核心思想：使用 $DFS$ 对图进行遍历，并为每个连通块尝试用两种颜色标记（例如 $0$ 和 $1$）。若在过程中出现相邻点颜色相同，则图不是二分图。

**伪代码（$DFS$ 版）**：

```
bool dfs(int u, int c) {
    color[u] = c;
    for (int v : G[u]) {
        if (!color[v]) {
            if (!dfs(v, c ^ 1)) return false;
        } else if (color[v] == color[u]) {
            return false;
        }
    }
    return true;
}

bool check() {
    for (int i = 1; i <= n; ++i)
        if (!color[i] && !dfs(i, 1))
            return false;
    return true;
}
```

**说明**：

- 若图不连通，需要对所有未染色点重复调用 `dfs`；

## 三、二分图的匹配与点覆盖

### 1. 最大匹配

**定义**：选出尽量多的边，使得任意两条边都不共享端点。
 当不存在**增广路**时，匹配即为**最大匹配**。

**常见算法**：

- 匈牙利算法（$DFS$ 增广，适合稀疏图，简单易写）
- $Dinic$ 算法（通过网络流模型实现）

### 2. 最小点覆盖

**定义**：选出尽量少的点，使得每条边至少有一个端点被选中。

二分图中的著名结论 $($$König$ 定理$)$

$最小点覆盖数=最大匹配数\text{最小点覆盖数} = \text{最大匹配数}最小点覆盖数=最大匹配数$

------

## 四、二分图的独立集与团

- **最大独立集**：选出最多的点，使得任意两点之间没有边。
- **最大团**：选出最多的点，使得任意两点之间都有边。

在二分图中：

$最大独立集大小=n−最大匹配数\text{最大独立集大小} = n - \text{最大匹配数}最大独立集大小=n−最大匹配数$

解释：
 删去最小点覆盖后剩下的点集即为独立集。

------

## 五、DAG 中的路径点覆盖

### 1. 最小路径点覆盖

**定义**：在一个有向无环图（DAG）中，用尽量少的**互不相交路径**覆盖所有点。

**做法**：

- 将原图中每个点拆成两个节点：入点 $u_{in}$ 与出点 $u_{out}$；
- 若存在边 $u \to v$，则连边 $u_{out} \to v_{in}$；
- 这样可得到一个二分图；
- 求该二分图的**最大匹配数 $m$**。

**结论**：

$最小路径点覆盖数=n−m\text{最小路径点覆盖数} = n - m最小路径点覆盖数=n−m$

------

### 2. 最小重复路径点覆盖

若路径允许重复（即可以多条路径经过同一顶点）：

- 首先对原图求**传递闭包**（例如用 $Floyd$ 算法）；
- 再在闭包图上重复上述拆点与匹配操作；
- 依然有：

$最小重复路径点覆盖数=n−m\text{最小重复路径点覆盖数} = n - m最小重复路径点覆盖数=n−m$

## 六、重要关系总结表

| 概念                         | 含义                 | 与匹配关系     | 计算方法           |
| ---------------------------- | -------------------- | -------------- | ------------------ |
| **最大匹配**                 | 选最多的互不相交边   | 基础           | 匈牙利 / Dinic算法 |
| **最小点覆盖**               | 选最少的点覆盖所有边 | = 最大匹配     | König 定理         |
| **最大独立集**               | 无边的最大点集       | = n - 最大匹配 | 剩余点             |
| **最小路径点覆盖 (DAG)**     | 不相交路径覆盖所有点 | = n - 最大匹配 | 拆点建二分图       |
| **最小重复路径点覆盖 (DAG)** | 路径可相交           | = n - 最大匹配 | 先求传递闭包再匹配 |
